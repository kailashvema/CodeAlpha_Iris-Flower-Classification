# Iris Flower Classification

## Project Overview
This project classifies Iris flowers into three species:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Algorithm
Random Forest Classifier

## Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Selection
5. Train-Test Split
6. Model Training
7. Model Evaluation

## Results
Achieved an accuracy of 96%.
